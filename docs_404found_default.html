404found 可以为我做什么？

404found 提供跨域网络加速服务，使用 OJBKSS，不仅可以让您轻松的访问那些「不太容易」访问到的网站，同时还加速所有服务器架设在境外的网站。从某种意义上来说，404found 还会相对增加你的网络传输安全性。当然，这一切都是基于完全不影响国内网站访问速度的前提下进行的。

我们为什么需要 OJBKSS？

在很久很久以前，我们访问各种网站都是简单而直接的，用户的请求通过互联网发送到服务提供方，服务提供方直接将信息反馈给用户

when evil come

然后有一天，GFW 就出现了，他像一个收过路费的强盗一样夹在了在用户和服务之间，每当用户需要获取信息，都经过了 GFW，GFW将它不喜欢的内容统统过滤掉，于是客户当触发 GFW 的过滤规则的时候，就会收到 Connection Reset 这样的响应内容，而无法接收到正常的内容

当然席不暇暖的我们很可能没有这份闲情逸致去感受大千世界、生活之美。但对于工作来说， 404found 将使你如虎添翼，更加轻松并出色的完成工作。
